# Sử dụng JS trong file HTML
- cài extension : Go live
- Taọ folder leson_1:
- Tạo 2 files trong folder lesson_1: index.html, main.js


## Hàm alert: 
- insert: "<script scr="./main.js"></script>" zô body của file main.js thì sẽ link đc file .js với file .html

- ở file main.js:
    alert('Hi Javascript basic!');
- Click Go live ở index.html => mở chrome và hiện message:"Hi Javascript basic!"
- Console : F12 -> chọn tab console
